JSON Toolkit’s desktop shell adds menu-driven workflows that complement the main editor UI.

## File menu actions (Mac app)

- **New Window** — `Shift+Cmd+N`
- **Open File…** — `Cmd+O`
- **Save As…** — `Cmd+S`
- **Open Diff Viewer** — `Shift+Cmd+D`
- **Anonymize to New Window** — `Shift+Cmd+A`

## Edit menu actions

- **Escape JSON**
- **Unescape JSON**

## Why this matters

- Open files into a fresh window so your current workspace stays intact.
- Compare data side-by-side with dedicated diff windows.
- Generate anonymized copies in new windows for safer sharing.
- Keep conversion tools available from the menu even when toolbar context changes.
